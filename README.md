# Имитатор звёздного неба

Приложение на C++/Qt для построения кадра звёздного неба в системе координат звёздного датчика КА.
В проекте реализованы звёзды каталога Hipparcos, Солнце, Луна и планеты Солнечной системы, а также интерактивные инструменты анализа кадра.

## Что реализовано

- Построение карты неба по параметрам наблюдателя и камеры:
  - центр визирования (RA/Dec),
  - поле зрения `FOV X/Y`,
  - углы поворота датчика `β1/β2/p`.
- Учёт эпохи наблюдения (дата из UI) для звёзд:
  - преобразование координат с учётом прецессии/нутации.
- Солнце считается отдельным эфемеридным модулем на дату наблюдения.
- Планеты: Mercury, Venus, Mars, Jupiter, Saturn, Uranus, Neptune.
- Луна:
  - эфемериды ELP82B (fallback на lite-провайдер),
  - фаза и ориентация терминатора,
  - топоцентрическая поправка (параллакс) в текущей модели наблюдателя.
- Визуальные эффекты:
  - Gaussian blur,
  - засветка/flare,
  - цветовой рендер и процедурные спрайты для Луны/части планет.
- Интерактивные возможности:
  - popup с данными по объекту при клике,
  - Zoom Inspector (включение зума, шаг зума, центрирование двойным кликом),
  - выделение выбранного объекта пунктирной окружностью.
- Сохранение кадра (`S` или `7` в виджете карты):
  - `save/sky_N.png`,
  - `save/sky_N.txt` с метаданными кадра.
- Режим Simulate:
  - выбор тела, диапазона дат и шага,
  - предрасчёт кадров с центрированием на выбранном теле,
  - проигрывание последовательности кадров в основном `MapWidget`.

## Управление

- `Построить` — пересчитать и отрисовать текущий кадр.
- `Доп Настройки` — blur/flare и режим размеров планет.
- `Симуляция` (`SimulateButton`) — запуск сценария предрасчёта/проигрывания кадров.
- В виджете карты:
  - `ЛКМ` по объекту — окно информации,
  - `Двойной ЛКМ` (в zoom-режиме) — центр зума,
  - `Z` — включить/выключить zoom,
  - `+/-` — шаг зума,
  - `Esc` — выход из zoom,
  - `S` или `7` — сохранить кадр.


## Сборка и запуск (qmake)

Требования:

- Qt 6 (`core`, `gui`, `widgets`, `openglwidgets`)
- C++17
- macOS/Linux с `qmake` и `make`

Сборка:

```bash
qmake starry_sky.pro
make -j4
```

Запуск:

```bash
./starry_sky.app/Contents/MacOS/starry_sky
```
